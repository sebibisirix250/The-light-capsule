1. Aplicația necesită instalarea soluției software XAMPP; se vor folosi modulele Apache și MySQL.
2. Copiați fișierele sursă (întregul folder „Website”) în directorul „htdocs” din folderul XAMPP.
3. Porniți o instanță locală a celor două componente.
4. Accesați http://localhost/Website/ pentru platformă și http://localhost/phpmyadmin/ pentru baza de date.
5. Pentru testarea funcționalităților din perspectiva unui utilizator, nu este necesară autentificarea. Pentru rolurile cu privilegii:
- client: email: andrei.cozahi@gmail.com, parolă: client;
- administrator: email: sebastian.ontijt@yahoo.com, parolă: admin.
6. Baza de date a fost exportată prin phpMyAdmin; creați o bază de date cu numele „website_shop” și folosiți același modul pentru import.
7. Se recomandă modificarea fișierului de configurare C:\xampp\php\php.ini pentru testarea funcționalităților de încărcare a conținutului visual, in special : "upload_max_filesize, post_max_size, max_file_uploads, max_execution_time"
8. Orice imagine inclusă în acest proiect reprezintă muncă proprie și conținut original; nu doresc publicarea acestora sub nicio formă.
9. Pentru accesarea de pe un dispozitiv mobil, este necesară aceeași conexiune la internet; utilizați adresa IP generată pe PC (prin comanda ipconfig) pentru a accesa link-ul sub forma [adresa_IP]/Website.
